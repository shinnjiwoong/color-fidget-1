CHANGE COLORS

*이 웹사이트는 모바일 기기에서 최적화된 디자인을 가지고 있습니다.

START버튼을 눌러 웹 사이트에 입장하실 수 있습니다.

GENERATE 버튼을 누르거나, 모바일 기기를 흔들 경우 눈의 색상이 랜덤하게 변합니다.

눈을 찌를 경우, 클립보드에 해당하는 색상의 색상 코드가 복사됩니다. 여러분의 프로젝트에 사용해보세요.

감사합니다.

*This website is optimized for mobile devices.

You can enter the website by clicking START button.

You can change the color of the eyes either by clicking GENERATE button, or shaking your device.

You can get the color code by poking the eye-ball. Use it on your project.
